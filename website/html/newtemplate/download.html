Download

To download the game, go here and click the download zip in the bottom right.

Once you have the file downloaded, unzip it.

Once the file is unzipped, go into the directory via terminal and make the file.

When that has finished compiling, go into the app directory and make that file as well.

After that has finished you may make r, or run the starclash file which was created.

When you're in the client, you will be able to view active lobies or play a test version of the game. The test mode is an offline single player version.
